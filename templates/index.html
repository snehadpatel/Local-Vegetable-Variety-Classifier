About the Project
VeggieSense Intelligence
Ensemble Deep Learning on Apple Silicon

VeggieSense is an advanced, hardware-accelerated classification engine designed to identify local Gujarati vegetable varieties with expert-level precision. At its core, the system processes input images by automatically stripping out complex backgrounds using a fast rembg model, leaving a clean, standardized black backdrop.

This processed image is then evaluated by an Ensemble of 4 Custom Convolutional Neural Networks. By combining predictions from a Global Average Pooling (GAP) CNN, a Multi-Scale CNN, a Residual CNN, and a Deep CNN, the model provides robust class probabilities. The final output is calibrated using temperature scaling and a Bayesian prior correction to deliver balanced, highly accurate predictions.
